## ðŸ’¡ Frequently Asked Questions

### For Program Directors

**Q: Does this replace wet labs or physical simulators?**
A: No. OphthalmoSim+ complements these resources by providing high-volume repetition between sessions. Think of it as daily practice between weekly lessons.

**Q: How do I track resident progress?**
A: The attending dashboard (Q1 2025) will provide real-time visibility into practice frequency, skill metrics, and comparative performance.

**Q: Is there evidence this improves surgical outcomes?**
A: Formal clinical trials are planned for 2026. Current evidence comes from internal pilot data and the broader simulation literature, which consistently shows skill transfer from virtual to real environments.

### For Residents

**Q: How much time should I spend on this?**
A: We recommend 15-30 minutes daily. Consistent short sessions outperform occasional long ones for motor skill development.

**Q: Will this help my OKAP scores?**
A: The OKAP module uses active recall methodology, which research shows is significantly more effective than passive review. Early users report improved confidence and practice scores.

**Q: Can I use this on my phone?**
A: The motor skill modules require a larger screen for accurate input. OKAP knowledge games work on phones, but we recommend a tablet for the full experience.

### For IT Administrators

**Q: What data do you collect?**
A: Performance metrics onlyâ€”no PHI, no video, no audio. All data is encrypted in transit and at rest.

**Q: Can this integrate with our LMS?**
A: LTI integration is on the roadmap for Q2 2025. Contact us for current integration options.

**Q: What's the uptime SLA?**
